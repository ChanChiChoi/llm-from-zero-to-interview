# 第二十册简介：Agent Harness、Coding Agent Runtime 与智能体工程框架

## 本书定位

第二十册专门讲 Agent Harness 和 Coding Agent Runtime。这里的 Harness 不只是 evaluation harness，而是更广义的大模型智能体运行框架，负责把模型、工具、技能、文件系统、终端、权限、安全策略、任务状态、日志追踪、评估回放和用户交互组织成可运行系统。

## 为什么需要这本书

Coding agent 不是一个简单模型 API。它需要读写文件、执行命令、编辑代码、管理上下文、申请权限、记录 trace、支持 replay 和评估任务成功率。这些能力都属于 runtime 和 harness 层。

## 适合读者

1. 想理解 Claude Code、OpenCode、Codex、Cursor、Aider、SWE-agent、OpenHands 等系统的人。
2. 想设计 coding agent、agent runtime 或 evaluation harness 的读者。
3. 准备 Agent 工程框架和系统设计面试的候选人。

## 学完后应具备的能力

1. 能解释 Agent Runtime、Tool Registry、Skill、Memory、Trace 和 Sandbox。
2. 能设计文件编辑、终端执行、权限确认和安全沙箱。
3. 能分析 Claude Code、OpenCode、Codex 等 coding agent 架构。
4. 能设计 evaluation harness 和 replay 机制。

## 阅读建议

建议先学习第十七册 Agent 基础，再读本书理解运行时工程。

## 与其他书的关系

第十七册讲 Agent 概念，第二十册讲 Agent 如何运行，第二十二册讲工具协议和生态接口。
