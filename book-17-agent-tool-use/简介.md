# 第十七册简介：Agent 与工具调用专题

## 本书定位

第十七册专门讨论 Agent、工具调用、规划、记忆、执行、Agentic RAG、code agent、browser/computer use、multi-agent、Agent 评估和安全边界。

## 为什么需要这本书

Agent 是把 LLM 从“回答问题”扩展到“执行任务”的关键方向。它涉及工具选择、任务分解、状态管理、外部执行、失败恢复、权限和安全。

## 适合读者

1. 想准备 Agent、tool use、function calling 和 code agent 方向的读者。
2. 已做过 RAG 或工具调用 demo，想系统化理解 Agent 架构的人。
3. 想设计可靠 Agent 系统和评估体系的人。

## 学完后应具备的能力

1. 能解释 ReAct、Plan-Act-Observe、planning 和 memory。
2. 能设计 tool calling、function calling 和 agentic RAG。
3. 能理解 code agent、browser agent、computer use agent 的风险。
4. 能评估 Agent 任务成功率、工具调用准确率和安全问题。

## 阅读建议

建议和第三册 RAG/Agent 项目、第十一册系统设计、第二十册 Agent Harness、第二十二册工具协议生态一起学习。

## 与其他书的关系

第十七册讲 Agent 概念和模式，第二十册讲 Agent Runtime，第二十二册讲工具协议和生态。
