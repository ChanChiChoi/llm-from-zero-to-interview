# 第二十一册简介：Transformer 架构详解、升级变种与未来架构演进

## 本书定位

第二十一册专门补齐 Transformer 架构深度、优缺点分析、现代 LLM 架构变体、Transformer 替代路线、主流大模型创新时间线和未来架构演进。

它覆盖原始 Transformer、现代 decoder-only LLM 改造、Linear Attention、SSM/S4/Mamba、RWKV、RetNet、Hyena、MoE、MLA、混合架构、主流模型演进和未来方向。

## 为什么需要这本书

第二册已经覆盖 Transformer 架构进阶，但仍属于总览。本书把“架构”作为独立专题，帮助读者真正理解为什么 Transformer 成为主流、瓶颈在哪里、Mamba/SSM 等路线想替代什么，GPT、LLaMA、Mistral、Mixtral、Gemini、Qwen、DeepSeek 等主流模型如何一步步互相借鉴和升级，以及未来架构更可能如何演进。

## 适合读者

1. 已理解基础 Transformer，但想深入架构 trade-off 的读者。
2. 准备模型架构深挖、论文讨论和开放研究题的候选人。
3. 想理解 Mamba、RWKV、RetNet、Hyena、Linear Attention 等路线的人。
4. 想按时间线理解主流大模型架构创新点和技术继承关系的人。

## 学完后应具备的能力

1. 能讲清 Transformer 的结构优势和 O(n^2) 瓶颈。
2. 能解释 RoPE、RMSNorm、SwiGLU、MQA/GQA、MLA、MoE 等现代改造。
3. 能比较 Linear Attention、SSM、Mamba、RWKV、RetNet、Hyena 等替代路线。
4. 能判断新架构论文的贡献、局限和工程可落地性。
5. 能按时间线讲清 GPT、LLaMA、Mistral、Mixtral、Qwen、DeepSeek 等模型的关键架构创新。
6. 能回答“Transformer 会不会被替代”这类开放题。

## 阅读建议

建议学完第一册 Transformer 主线和第二册架构进阶后阅读。本书适合配合论文精读、主流模型技术报告和架构开放题训练。

## 与其他书的关系

第二十一册是第二册架构部分的专题扩展，也为第十册论文研究、第十六册 reasoning 架构和第十五册多模态架构提供支撑。
