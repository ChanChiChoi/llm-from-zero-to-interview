# 第七册简介：大模型评估、实验与科学方法

## 本书定位

第七册专门回答一个核心问题：你怎么知道模型真的变强了？

它覆盖 benchmark、human eval、pairwise eval、LLM-as-Judge、污染检测、reasoning/code/math eval、多模态评估、安全评估、在线实验、统计显著性和 error analysis。

## 为什么需要这本书

大模型能力复杂，单一分数很容易误导判断。没有可信评估，就无法判断训练、微调、RAG、DPO、量化或上线实验是否真的有效。

## 适合读者

1. 想准备模型评估、实验平台、算法研究和上线验证岗位的读者。
2. 经常需要比较两个模型版本的人。
3. 想提升实验设计和 bad case 分析能力的候选人。

## 学完后应具备的能力

1. 能设计 benchmark 和人工评测表。
2. 能使用 pairwise preference、LLM-as-Judge 和人工 gold set。
3. 能识别 benchmark contamination 和评估污染。
4. 能做统计显著性、A/B 测试和上线门禁。
5. 能用 error attribution 定位模型、数据、检索、prompt 或系统问题。

## 阅读建议

建议在做第三册评估 Debug、RAG 项目和模型上线实验时同步阅读。

## 与其他书的关系

第七册是全书系的实验方法基础。训练、部署、RAG、Agent、多模态、安全和产品化都依赖可信评估。
