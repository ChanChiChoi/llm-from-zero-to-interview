# 第三册简介：大模型算法岗实战手册

## 本书定位

第三册是实战训练书，目标是把第一册和第二册中的理论知识转化为可运行代码、可复现实验、可展示项目和可用于面试表达的技术成果。

它不是单纯代码合集，而是围绕“算法岗面试中的实战可信度”设计：能手写核心组件，能跑通训练和微调，能搭建 RAG/Agent/推理服务，能设计评估和 debug，能把项目讲成高质量面试故事。

## 为什么需要这本书

很多候选人能背概念，但项目深挖时讲不清数据、loss、指标、bad case 和工程取舍。第三册解决这个问题：每个主题都尽量落到代码、实验、评估、简历表达和面试追问。

## 适合读者

1. 已经开始学习大模型原理，希望通过代码巩固理解的读者。
2. 准备简历项目和项目深挖的候选人。
3. 想系统补齐 PyTorch、Hugging Face、LoRA、DPO、RAG、推理服务和多模态实战的人。

## 学完后应具备的能力

1. 能用 PyTorch 实现关键深度学习和 Transformer 组件。
2. 能从零训练小 GPT，并分析 loss 和生成样例。
3. 能使用 Hugging Face、PEFT、TRL 完成 SFT、LoRA、QLoRA 和 DPO 实验。
4. 能搭建 RAG、Agent 和高性能推理服务 demo。
5. 能设计人工评测、A/B 测试、bad case 分析和 debug 清单。
6. 能完成 miniGPT、中文助手、DPO、RAG、推理服务、数学推理和多模态项目作品集。

## 内容结构

1. PyTorch 基础实战。
2. Transformer 组件实战。
3. 从零训练小 GPT。
4. Hugging Face 微调实战。
5. 偏好优化实战。
6. 推理优化实战。
7. RAG 与 Agent 项目实战。
8. 评估与 Debug 实战。
9. 简历项目实战。
10. 面试实战训练。
11. 多模态实战。

## 阅读建议

建议边读边运行代码。每完成一个部分，都要沉淀一个可以讲给面试官的项目故事：背景、目标、方案、指标、结果、bad case 和下一步改进。

## 与其他书的关系

第三册是整套书系的“动手层”。第一册和第二册负责讲原理，第三册负责把原理变成代码和项目，第十二册会进一步把这些项目包装成求职材料。
