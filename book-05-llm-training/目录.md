# 第五册：大模型训练全流程

## 本书定位

本书专门介绍大模型训练相关知识，覆盖从数据、tokenizer、模型架构、预训练、后训练、对齐强化训练、分布式训练、训练稳定性、评估、实验管理到训练事故排查的完整流程。

第一册和第二册会讲训练的核心原理，但本书更偏“训练全流程”和“工程实战思维”，目标是让读者能回答：如果让你负责训练一个大模型，你会如何设计、执行、监控、评估和 debug。

## 核心问题

1. 如何设计大模型训练数据 pipeline？
2. 如何选择 tokenizer、模型规模和训练 token 数？
3. 如何组织预训练、SFT、RLHF/PPO、DPO、GRPO、GSPO、DAPO、DCPO 和多模态训练？
4. 如何选择 AdamW、Adafactor、Lion、Sophia、Muon 等优化器和学习率策略？
5. 如何做分布式训练和显存优化？
6. 如何监控训练是否健康？
7. 如何处理 loss spike、NaN、梯度爆炸、吞吐低、评估异常？
8. 如何估算训练成本和资源需求？
