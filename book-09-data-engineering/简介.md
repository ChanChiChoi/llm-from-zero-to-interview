# 第九册简介：大模型数据工程与数据智能

## 本书定位

第九册专门讲大模型数据，目标是理解数据如何决定模型能力，以及如何构建高质量、可治理、可评估的数据体系。

## 为什么需要这本书

大模型能力不是只由架构决定，数据质量、数据配比、去重、污染控制、合成数据和偏好数据都会决定最终行为。许多训练和评估问题，本质上都是数据问题。

## 适合读者

1. 想准备数据工程、训练数据、后训练数据和评估数据相关岗位的读者。
2. 需要构建 SFT、DPO、RAG、多模态或安全数据集的人。
3. 想理解数据价值评估和数据治理的人。

## 学完后应具备的能力

1. 能设计 web-scale 数据采集、清洗、过滤和去重流程。
2. 能分析 benchmark contamination 和数据泄漏。
3. 能设计 data mixture、代码/数学/领域数据和多模态数据配比。
4. 能构造 preference data、安全数据和合成数据。
5. 能做 dataset versioning、governance 和 data valuation。

## 阅读建议

建议和第五册训练全流程、第七册评估、第八册安全一起学习。

## 与其他书的关系

第九册是训练、对齐、评估、RAG、多模态和安全专题的数据基础。
