# 第十四册简介：PyTorch 与深度学习工程

## 本书定位

第十四册训练工程实现能力，帮助读者从“能看懂代码”升级到“能写训练脚本、能调试模型、能实现核心组件、能组织实验工程”。

## 为什么需要这本书

大模型算法岗不只考理论，还会考能否手写 attention、loss、训练循环、mask、debug、profiling 和分布式基础。PyTorch 是把理论变成实验的主要工具。

## 适合读者

1. PyTorch 不熟、训练脚本写得不稳的读者。
2. 想准备手写代码和工程 debug 面试的人。
3. 想为第三册实战打好工程基础的人。

## 学完后应具备的能力

1. 能熟练操作 tensor、autograd、nn.Module、Dataset 和 DataLoader。
2. 能写完整训练循环、验证循环、checkpoint 和日志。
3. 能理解混合精度、显存优化、分布式训练入门。
4. 能实现 Transformer 核心组件并排查 shape、grad 和 loss 问题。

## 阅读建议

建议和第三册前两部分同步学习，边看边写代码。

## 与其他书的关系

第十四册是第三册实战、第十三册数学和第五册训练系统之间的工程桥梁。
